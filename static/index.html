How does it work through my firewall?
With WebRTC and STUN, two protocols that can punch holes through network address translators. This is the same technique used by apps like Skype and Xbox Live.
Why is there a speed limit?
Unfortunately, technical limitations. WebRTC is not a very efficient way to move lots of data as quickly as possible.
